Welkom!

Dit is een klein voorbeeld van een webapplicatie in Flask

Je hebt zojuist het commando 

docker run -d -p 5000:5000 webservercontainer
uitgevoerd. Benieuwd naar wat dat allemaal betekent?

  • docker - het container-programma dat we binnen deze VM gebruiken
    • run - we willen een container die al gebouwd is uitvoeren.
    • -d - detached: doe dit op de achtergrond (zodat je commandline weer terugkomt en je nieuwe commando's kunt uitvoeren)
    • -p 5000:5000 - binnen de container hebben we webserver-poort nummer 5000 opengezet. Die willen we ook op onze host kunnen benaderen onder hetzelfde poortnummer. In je webadres zie je dit terug in de tekst 
      :5000
    • webservercontainer - de tag van de container. Deze heb je opgegeven toen je 
      docker build
      uitvoerde
